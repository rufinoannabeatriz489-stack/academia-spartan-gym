# Spartan Gym — Site estático

Arquivos incluídos:
- index.html
- styles.css
- script.js
- assets/
  - hero-1.jpg   (imagem 3 - esteira)
  - natacao.jpg  (imagem 2 - natação)
  - product-1.jpg (imagem 1 - suplemento)
  - map-placeholder.png (imagem genérica de mapa)
- robots.txt
- CNAME

Como usar:
1. Coloque as imagens enviadas na pasta `assets/` com os nomes indicados acima.
2. Abra `index.html` no navegador para testar localmente.
3. Para publicar: suba todos os arquivos para GitHub Pages, Netlify, Vercel, ou outro host estático.
   - GitHub Pages: crie um repositório, faça commit dos arquivos e ative Pages.
   - Netlify/Vercel: arraste a pasta para fazer o deploy ou conecte ao repositório.
4. Integrações recomendadas:
   - Substituir o `alert()` do formulário por uma chamada ao seu backend (/api/contact) ou usar um serviço como Formspree/Zapier.
   - Integrar Google Maps na seção de localização (substituir map-placeholder.png por iframe).
   - Para vendas, integrar Stripe/PagSeguro ou criar páginas de produto com checkout.

Contato usado no site:
- Telefone: (19) 99999-9999
- Email: contato@spartangym.com

Se quiser, eu posso:
- Gerar o arquivo ZIP com tudo pronto (você fará o download) — eu preparo o ZIP e disponibilizo no próximo passo.
- Converter para um template React/Next.js ou WordPress.
- Integrar formulário com um endpoint real ou com WhatsApp click-to-chat.
